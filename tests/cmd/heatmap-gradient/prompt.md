# Heatmap Gradient Test

## Expected Visual Output

A 20x20 heatmap displaying:
- A radial gradient pattern
- Center showing highest values (brightest/warmest color)
- Edges showing lowest values (darkest/coolest color)
- Smooth color transition from center to edges

## Verification Criteria

1. **Color Gradient**: Visible color variation across the heatmap
2. **Radial Pattern**: Concentric color bands emanating from center
3. **Center Bright**: Center area shows maximum value color
4. **Edge Dark**: Corner areas show minimum value color
5. **Smooth Transition**: No abrupt color changes

## Pass Conditions

- Multiple distinct colors visible
- Center is visually distinct (lighter/warmer) from edges
- Circular/radial pattern evident
- Grid fills the display area
