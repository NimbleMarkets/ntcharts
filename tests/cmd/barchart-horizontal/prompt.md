# Horizontal Bar Chart Test

## Expected Visual Output

A horizontal bar chart with 4 bars displaying:
- Bar "A" with value 80 (longest bar)
- Bar "B" with value 60
- Bar "C" with value 40
- Bar "D" with value 20 (shortest bar)

## Verification Criteria

1. **Orientation**: Bars extend horizontally from left to right
2. **Labels**: Each bar has a label (A, B, C, D) on the left side
3. **Proportions**: Bar lengths should be proportional (A is 4x longer than D)
4. **Colors**: Bars should have distinct visible colors
5. **Axis**: Y-axis showing bar labels, X-axis showing values

## Pass Conditions

- 4 horizontal bars visible with correct relative lengths
- Labels A, B, C, D visible
- Bar A is longest, D is shortest
- Colors distinguish the bars
