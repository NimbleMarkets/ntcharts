# Waveline Sine Pattern Test

## Expected Visual Output

A waveline chart (30x12) displaying:
- Y-axis with values from -3 to 3
- X-axis with values from 0 to 10
- Points connected with wave-style curves: (1,2), (3,-2), (5,2), (7,-2), (9,2)
- Smooth wave connections between points (not straight lines)

## Verification Criteria

1. **Wave Connections**: Points connected with curved/wave segments, not straight lines
2. **Alternating Pattern**: Values alternate between +2 and -2
3. **Axes**: Both X and Y axes visible with labels
4. **Symmetric**: Pattern is regular and symmetric

## Pass Conditions

- Clear alternating up/down wave pattern
- Smooth wave curves connecting points
- Y-axis shows range including -2 to 2
- Five peaks/valleys visible
