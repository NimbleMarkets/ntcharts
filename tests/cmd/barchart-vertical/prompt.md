# Vertical Bar Chart Test

## Expected Visual Output

A vertical bar chart with 4 columns displaying:
- Column "A" with value 80 (tallest)
- Column "B" with value 60
- Column "C" with value 40
- Column "D" with value 20 (shortest)

## Verification Criteria

1. **Orientation**: Bars extend vertically upward from the bottom
2. **Labels**: Each column has a label (A, B, C, D) at the bottom
3. **Proportions**: Column heights should be proportional (A is 4x taller than D)
4. **Colors**: Columns should have distinct visible colors
5. **Axis**: X-axis showing column labels, Y-axis showing values

## Pass Conditions

- 4 vertical columns visible with correct relative heights
- Labels A, B, C, D visible at bottom
- Column A is tallest, D is shortest
- Colors distinguish the columns
