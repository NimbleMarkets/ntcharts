# Time Series with Dates Test

## Expected Visual Output

A time series chart (50x12) displaying:
- Y-axis with values from 0 to 100
- X-axis with date labels
- A line showing values over 7 consecutive days: 20, 40, 35, 60, 80, 70, 90

## Verification Criteria

1. **Date Labels**: X-axis shows date values (may be abbreviated)
2. **Value Labels**: Y-axis shows numeric values
3. **Data Points**: Line connects the data points showing upward trend with variation
4. **Time Ordering**: Points are ordered chronologically left to right

## Pass Conditions

- Both axes have labels
- Line shows general upward trend
- Date formatting visible on X-axis
- Values rise from ~20 to ~90 with mid-chart dip
