# Streamline Wave Pattern Test

## Expected Visual Output

A streaming line chart (30x10) displaying:
- Y-axis with values from 0 to 10
- A wave pattern that streams from right to left
- Values: 4, 6, 8, 10, 8, 6, 4, 2, 0, 2, 4 creating a sine-like wave
- Most recent values on the right side

## Verification Criteria

1. **Wave Shape**: Clear sinusoidal pattern visible
2. **Axes**: Y-axis visible with numeric labels
3. **Peak and Valley**: Line reaches maximum (10) and minimum (0)
4. **Connected Line**: Continuous line connecting all points

## Pass Conditions

- Wave pattern clearly visible
- Y-axis shows scale from 0 to 10
- Line shows peak around 10, valley around 0
- Smooth connected line segments
